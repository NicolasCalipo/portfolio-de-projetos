Os Meus Vídeos de Curiosidades Favoritos no Niflix!

Imagina um lugar no Niflix onde você pode mostrar todos os teus vídeos de curiosidades favoritos de uma forma super bacana! É exatamente isso que este projeto faz: ele cria uma página da web personalizada, usando HTML e CSS, para exibires a tua coleção.

Nessa página, cada vídeo tem uma descrição rápida, o canal do criador, o ano em que foi lançado e até um link para assistires e partilhares com os teus amigos! Tudo foi pensado para ser fácil de ver em qualquer tela, seja no computador ou no telemóvel, e tem um visual suave e convidativo, com cores e fontes escolhidas a dedo para te sentires à vontade enquanto exploras a tua própria coleção virtual.

É como ter um pedacinho especial do Niflix só para os teus vídeos de curiosidades preferidos!
